# GameNightAPI

The aim of this project is to help users quickly select an appropriate board game based on number of players, available time, mood, age, and difficulty
## Practical Significance
- Reduces time spent deciding what game to play.
- Useful for casual players, families, students, and board game cafes.
- Provides you with a list of recommended games according to your preferences.
## Technologies Used

- **C#**: Primary language for backend development.
- **JavaScript**: Used for frontend logic.
- **HTML**: Markup for the user interface.
- **CSS**: Styles for the application.
- **.NET/ASP.NET Core**: Framework for building the API.

## Installation

To get started with GameNightAPI, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/naskeriya/GameNightAPI.git
   ```
2. Navigate to the project directory:
   ```bash
   cd GameNightAPI
   ```
3. Restore the dependencies:
   ```bash
   dotnet restore
   ```
4. Run the application:
   ```bash
   dotnet run
   ```
